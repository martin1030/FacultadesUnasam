<h5 align="center"><b>SISTEMA GESTOR DE CONTENIDO DE PAGINAS WEB PARA LAS FACULTADES DE LA UNASAMA</b></h5>
<p align="justify">El sistema esta desarrallado en PHP Y VUEJS usando el Framework LARAVEL el sistema es un gestor de contenido para todas la paginas de las facultades de la unasam el sistema tiene un modulo cliente que puede acceder cualquier usuario y el modulo de administrador que solo tiene ingreso aquellas personas que cuentan con un usuario y su tipo de usuario ademas la autenticacion trabaja con roles con este metodo el sistema tiene mejor seguridad y se puede configurar los roles para cada usuario, el sistema trabaja con VUEJS</p>
